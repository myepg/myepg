### MYEPG 사용법
  1. A1 사용자는 ON
  2. Proxy 사용자는 OFF
  3. Proxy 는 support_site - wavve - proxy 활성화 하고 입력
  4. 원하는 소스 선택 
    - SPOTV 소스로 테스트 추천, 채널개수가 적어서 금방 완료됩니다.
  5. 설정 저장
  6. 즉시 실행 or 1회 실행
    - Channel.json 를 생성/업데이트 하거나 xmltv.xml 만들 때 오래 걸립니다.
  7. 테스트 마쳤으면 스케쥴링 설정 하고 api 입력
